# North Star

## Product one-liner

A smooth Pyxel prototype where the player observes expressive fireworks in a small cutout space.

## Current product goal

freely observing fireworks from changing angles

## Primary value

The project should make the player feel:

- smooth, beautiful, surprisingly rich for Pyxel
- relaxing and visually satisfying

## Must not lose

smooth visual feel and small maintainable implementation

## Success criteria

The prototype succeeds when:

- a playable prototype with satisfying fireworks, stable frame pacing, and clear Codex task continuity
- the core loop is playable,
- visual direction is recognizable,
- Codex can continue development from repository docs rather than hidden chat context.
