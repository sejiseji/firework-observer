# Pyxel Regression Review Skill

Use this after Codex changes before committing.

## Review

Check:

- Does the change match the stated goal?
- Did it modify unrelated systems?
- Are tests present for pure logic?
- Are visual changes smoke-tested?
- Are risks and follow-up tasks recorded?

## Report format

- Summary
- Files changed
- Tests run
- Behavior verified
- Risks
- Follow-up tasks
